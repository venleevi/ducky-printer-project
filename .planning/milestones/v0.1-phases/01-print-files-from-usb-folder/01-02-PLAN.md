    2. **find_printer() function** - Auto-detect USB printer by class:
       ```python
       def find_printer():
           """Auto-detect USB thermal printer by device class (class 7)."""
           import usb.core
           from escpos.printer import Usb

           def match_printer_class(device):
               # Check device-level class
               if device.bDeviceClass == 7:
                   return True
               # Check interface descriptors
               try:
                   for cfg in device:
                       for intf in cfg:
                           if intf.bInterfaceClass == 7:
                               return True
               except:
                   pass
               return False

           try:
               printer = Usb(usb_args={'custom_match': match_printer_class})
               return printer
           except Exception as e:
               raise PrinterError(f"No printer found: {e}")
       ```

    3. **print_text(content: str) function** - Print text with per-job lifecycle:
       - Call find_printer() to get printer instance
       - Open connection: printer.open()
       - Wrap in try/finally to ensure cleanup
       - Feed 1 blank line: printer.feed(1)
       - Print content: printer.text(content)  # python-escpos handles UTF-8 via MagicEncode
       - Feed 2 blank lines: printer.feed(2)
       - Full cut: printer.cut(mode='FULL')
       - Close connection in finally block: printer.close()
       - Catch usb.core.USBError → close and raise PrinterError("USB error: {e}")
       - Catch generic Exception → close and raise PrinterError("Printer error: {e}")
       - Return 0 on success
